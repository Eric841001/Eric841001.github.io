# Knowledge Center

Microsoft Enterprise Consulting Platform의 핵심 기술 지식 허브입니다.

이 영역은 단순 기술 정리가 아니라, 실제 고객 제안·설계·구축·운영에서 바로 재사용할 수 있는 Enterprise Knowledge Base로 구성됩니다.

## Recommended Starting Points

### Microsoft 365

Microsoft 365 영역은 기업의 협업, 보안, 거버넌스, 라이선스 최적화를 다룹니다.

- E3 vs E5 비교
- SharePoint / OneDrive Governance
- Teams 운영 정책
- Exchange Online 보안 정책
- Conditional Access 설계
- Purview Information Protection

### Security

Security 영역은 Zero Trust 기반의 Identity, Device, Data, Access Control 설계를 다룹니다.

- Entra ID
- Conditional Access
- Intune
- Defender
- Purview DLP
- Sensitivity Label
- MIP / Information Protection

### Copilot

Copilot 영역은 기술 배포뿐 아니라 Adoption, Change Management, Governance까지 포함합니다.

- Copilot Readiness
- Data Security Readiness
- User Enablement
- Champion Program
- Prompt Library
- Adoption KPI

### Migration

Migration 영역은 고객 환경 전환 시 필요한 실무 기준을 제공합니다.

- Google Workspace to Microsoft 365
- Tenant to Tenant Migration
- NAS / File Server to SharePoint
- Exchange Hybrid Migration
- Domain Cutover
- Migration Risk Management

## Quality Standard

이 Knowledge Center는 다음 기준을 따릅니다.

- Microsoft Learn 수준의 기술 정확성
- Microsoft Consulting 수준의 문서 구조
- 실제 고객 프로젝트에 적용 가능한 실무성
- 라이선스 포함 여부와 제약사항 명확화
- 보안·운영·비용 관점 동시 반영
- 고객명은 사용하지 않고 산업군 기준으로 일반화
